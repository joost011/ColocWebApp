Welcome to ColocAPI

Coloc API allows for colocalization analysis to compare genetic associations datasets (e.g. GWAS and eQTL). It returns summary files and locus visualization plots to allow for detailed review of the results. The summary files are downloadable. For the eQTL association dataset, data from the eQTL consortium phase I is used. It contains summary statistics of 16.989 cis eQTL genes from 31.684 blood samples, making it the biggest eQTL dataset to date. More eQTL datasets are planned soon to be added. With coloc API you are able to upload your own GWAS summary statistics for the colocalization analysis. It supports human datasets in build GRCh37 or GRCh38. Different phenotypes (e.g. quantitative and case-control) for GWAS studies can be processed. For colocalization analysis, the coloc package is used to see if there is a common genetic causal variant in a given region. It is also possible to use the SuSIE approach to deal with multiple causal variants.
